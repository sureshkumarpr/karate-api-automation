📋 Step-by-Step Maven Execution

1
mvn clean
Cleans the project by removing the target directory and any compiled files
✅ Clean completed successfully
2
mvn compile
Compiles the source code and creates the target/classes directory
✅ Source compilation completed
3
mvn test-compile
Compiles the test code and creates the target/test-classes directory
✅ Test compilation completed
4
mvn test
Executes all tests and generates reports in target/surefire-reports and target/karate-reports
✅ All tests executed successfully (15/15 PASSED)
